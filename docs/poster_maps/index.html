What is this portal?

This portal hosts interactive network maps and model outputs from the ePiE (Exposure to Pharmaceuticals in the Environment) project, modelling pathogen fate and transport in river basins.

GitHub repository | Poster site

Available outputs include:

  • Interactive concentration maps — explore pathogen spatial distribution and colored reaches
  • Interactive network maps — tap nodes for discharge, population, and source data
  • Data tables — CSV files with raw model results
  • Static maps & GIS layers — assets for reports and spatial analysis
